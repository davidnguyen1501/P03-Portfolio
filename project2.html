KINETIC TYPOGRAPHY

Overview

I Worked on this project from a Graphic Design course (IAT100) alone and created everything from start to finish using the Adobe Suite, namely Illustrator, After Effects and Photoshop. The goal of this project is to create a Kinetic Typography video that features a lot of animation so that each student can express their skills in using basic graphic design software as well as their ability to be creative

Process

Ideation Process

To start off with the project, I had to pick a song that has a lot of words that I can turn into a lively animation, then brainstorming a storyboard based on what I think would fit best
